1. Typography & Text Elements

Tag: <p> and <br> - The paragraph tag groups text blocks. The <br> tag creates a manual line break without starting a new paragraph.

Tag: <abbr> - Used for abbreviations. The title attribute shows the full text when you hover over the dotted underline.

Tag: <strong> and <em> - <strong> is for high importance (bold), while <em> is for stress emphasis (italics).

Tag: <b>, <i>, and <u> - These are stylistic tags. <b> is bold without extra importance, <i> is italics for alternate voice, and <u> is an unarticulated underline.

Tag: <del>, <ins>, and <s> - <del> shows deleted text (usually with a line through it), <ins> shows added text (usually underlined), and <s> shows text that is no longer accurate.

Tag: <blockquote>, <q>, <cite> - <blockquote> is for long external quotes. <q> is for short inline quotes. <cite> identifies the title of the work cited.

2. Media & Multi-media Elements

Tag: <figure>, <figcaption>, <picture>, <source>, <img> - <figure> holds media and its caption. <picture> allows you to provide different image versions (<source>) for different screen sizes before falling back to the <img>.
Tag: <map>, <area> - These create a "Clickable Image Map". You define specific coordinates (shapes) on an image that link to different places.
Tag: <canvas>, <svg> - <canvas> is a box for Javascript-drawn graphics. <svg> is for vector graphics (math-based shapes).

Tag: <audio>, <video>, <track>, <iframe>, <object> - Tags for playing sound, movies (with <track> for subtitles), and embedding external websites or resources.

5. Forms & Interactive Inputs

Tag: <form>, <fieldset>, <legend> - The <form> handles the data. <fieldset> groups related inputs with a border, and <legend> provides a title for that group.
User Preferences
Tag: <label>, <input> - Always connect <label> to an <input> using the for and id attributes for accessibility.
Tag: <datalist> (Suggestion Link) - This is how auto-suggestions work. You link an <input> to a <datalist> using the list attribute. Users can still type anything, but they see these options as they type.
Tag: <optgroup>, <option> - Use <optgroup> inside a dropdown (<select>) to categorize your <option> choices.
Tag: <output> - Displays the result of a calculation (like the range slider below moving).

Tag: <textarea>, <button> - <textarea> for multi-line text. <button> for triggers (Submit, Reset, etc.).
